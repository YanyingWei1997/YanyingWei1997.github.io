# Repository Memory

## Cover Image Style

When generating or editing blog/publication cover images for this repository, match the existing visual language from:

- `images/blog-stata-ai.png`
- `images/blog-literature-review.png`
- publication cards such as `images/carbon-peak-uncertainty.png`

Required style:

- 16:9 horizontal cover, preferably `1200x675` or similar.
- Clean white or very pale blue background with soft abstract waves.
- Deep navy main color: `#012F63` / `#082C62`.
- Coral accent color: `#FE667B`.
- Pale blue and blue-gray support colors.
- Thin navy outlines, rounded UI/document cards, soft shadows.
- Dotted connector lines and small dotted decorative accents are acceptable.
- Use one short English title in the upper-left only when useful.
- Avoid long in-image text; the website card already shows the full article title.
- Use topic-specific academic/tool objects: laptop, documents, magnifying glass, chart, database, verified checkmark, Zotero/library panel, LaTeX/BibTeX cue.
- Keep the composition airy and polished: one central object, two or three supporting objects, generous whitespace.

Avoid:

- Dark dashboards.
- Dense boxed flowcharts.
- Saturated gradients.
- Heavy shadows.
- Generic stock-photo style.
- Random decorative blobs.
- Large Chinese title text inside the image.

Always compare the new cover against `blog-stata-ai.png` and `blog-literature-review.png` before accepting it.
